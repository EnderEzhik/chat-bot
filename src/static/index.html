Доступные команды

Шаблоны сообщений:

  • Проблема с подключением - "У меня не работает интернет"
  • Сброс пароля - "Забыл пароль от учетной записи"
  • Обновление системы - "Как обновить операционную систему?"

Команды:

  • /clear - очистить историю чата
  • /help - показать справку
  • /theme [light/dark] - сменить тему
  • /history - показать историю сессии
